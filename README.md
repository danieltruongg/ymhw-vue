# ymhw-website

## Project Setup

```sh
// Generate node_modules within root
npm install

// Generate node_modules within functions
cd functions
npm install
```

## Firebase Configuration

1. Setup a firebase with the following features:
   - Email & Password Authentication enabled
   - Firestore Database configured
   - Firebase Functions enabled
2. Following the example .env file, configure your own .env file generated by your firebase (root dir)
3. Following the example .env file, configure your own .env file for your firebase functions (functions dir)

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
